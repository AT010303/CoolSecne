# Cool Scene

## Setup

To run this project locally, download [Node.js](https://nodejs.org/en/download) and follow these steps:

```
# Install dependencies
npm install

# Run the local server at localhost:5173
npm run dev

# Build for production in the dist/ directory
npm run build
```
